Swiss Cheese Project

About Swiss Cheese Project

Swiss Cheese Project is a hands-on, interactive web application security Crackathon.

I'm in a Session now! (Let's get started)

Overview

Swiss Cheese Project is a Deliberately Vulnerable Web Application. It's designed to give people an opportunity to investigate, probe and discuss fixes for, common web application vulnerabilities

History

The first Swiss Cheese Project session was run at Software Craftsmanship 2013 (yes that's you!!)

Platform

Swiss Cheese Project runs on the (fictional) Edam Web Server (but is, in actual fact, a basic Dropwizard app with a Mustache and Twitter Bootstrap front-end and HSQLDB ... you can't have a vulnerable app without a bit of SQL)
